# 🍳 Chef Cooking AI Agent

Chef Cooking AI Agent is an AI-powered cooking assistant that helps users cook recipes step-by-step.  
The system uses **FastAPI, LangGraph, and LLMs** to simulate a friendly chef that guides users through cooking instructions interactively.

---

## 🚀 Features

- 👨‍🍳 AI Chef personality
- 🧠 Multi-agent workflow using **LangGraph**
- 💬 Interactive cooking conversation
- 🍛 Dynamic recipe generation
- 📜 Conversation history
- ⚡ FastAPI backend
- 🎨 HTML interface using **Jinja Templates**
- 🔐 Secure API key management with `.env`
